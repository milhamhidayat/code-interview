# Triangle1

Tulis sebuah function dimana menerima input angka N positif. Function akan print segitiga dengan N level menggunakan karakter *. Pastikan pada bagian kanan segitiga memiliki space kosong.

Contoh:

```
Level 2

'* '
'**'
```

```
Level 3

'*  '
'** '
'***'
```

```
Level 5

'*    '
'**   '
'***  '
'**** '
'*****'
```
