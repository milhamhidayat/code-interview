# Flip Triangle

Tulis sebuah function dimana menerima input angka N positif. Function akan print segitiga terbalik dengan N level menggunakan karakter *. Pastikan pada bagian kiri segitiga memiliki space kosong.

Contoh:

```sh
Level 2

' *'
'**'
```

```sh
Level 3

'  *'
' **'
'***'
```

```sh
Level 5

'    *'
'   **'
'  ***'
' ****'
'*****'
```
