# Matrix Spiral

Tulis sebuah function yang menerima integer N dan kembalikan matrix sprial N x N. Tiap slot dalam matrix diisi dengan angka berurutan dan diisi secara searah jarum jam.
Contoh:

```sh

N = 2
output =

[
    [1,2],
    [4,3],
]

N = 3
output =

[
    [1, 2, 3],
    [8, 9, 4],
    [7, 6, 5],
]

```
